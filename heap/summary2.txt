

heap 
	: 루트의 값만 유효함
	: compelete binary tree 구조
	: 종류
		: max heap : 부모가 두 자식값 이상인 구조. >> 루트에 최댓값 

		: min heap : 부모가 두 자식값 이하인 구조. >>  루트에 최솟값 


heapify : 자료구조를 힙화 시키는것. 
	: 목적 >> 빠른 속도로 최소/대 값 구하기 위함
	: 부모노드 들만 힙화 시킨다?? (리프 노드들은 x) 
	: maxheap 만드는 법
		1. 자식 노드끼리 값을 비교하여 더 큰 값 get
		2. 해당자식노드 > 부모이면 swap 
		3. 1~2 를 반복.


heap 에 원소 추가
	: maxheap 에서
		1. 일단 맨 끝 추가
		2. 부모와 자식 비교 >> 자식이 더 크면 swap
		3. 1~2 를 반복

heap 에서 원소 삭제
	: 루트값을 추출하는 것
	: maxheap 에서
		1. 마지막 레벨 젤 끝의 값을 루트노드로 복사
		2. 해당 마지막 노드를 삭제
		3. 루트 노드와 자식 노드 비교 > 루트가 더 작으면 swap
		4. 부모와 자식 비교 > 부모가 더 작으면 swap
		5. 4를 반복
