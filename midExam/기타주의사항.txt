1. main 메서드도 Static 이기 때문에 내부에서 인스턴스 변수, 메서드 사용 및 호출 불가하다
    : main 메서드 내부서 직접 사용하고 싶으면 필수적으로 해당 필드 혹은 메서드를 static 으로 선언해줘야된다

2. 배열 객체 생성에서 사이즈 명시하는건 new 타입[n] 밖에 없다
3. new 배열[ ]{값1, 값2 , .. }는 선언 이후에도 할당 가능하지만 , {값1, 값2 .. } 는 할당 불가하다

4. 가변 배열은 "enw 배열명[열사이즈][ ]" 로 만든다  (행사이즈를 생략한다 ) && 이후 배열명[i] 별로 일차원배열할당해서 사용한다
        : 생각해보면 들쑥 날쑥한건 행 사이즈지 열 사이즌 아니니까 

5. 클래스는 1개만  extends 가능하다
    : 그런데도 모든 클래스가 자동적으로 Object 클래스를 부모로 하는 이유는 extends 윗부분에서 누군가가 Object 를 부모로 하기 때문
