113. 컴파일러의 구성 요소는 ? 
  : front end 부분 >> 
    (1) lexical analyzer : 어휘 분석기 --> 토큰 생성
    (2) syntax anaylzer : 구문 분석기 --> 파싱. 그러니까 토큰의 관계를 파악하여 파스 트리 생성
    (3) semantic analyzer : 의미 분석기 --> 파스 트리를 바탕으로 중간 표현(Itermediate Expression) 생성
  : back end 부분 >> code generator : 코드 생성기 --> 중간 표현을 바탕으로 어셈블리어(중간언어) 혹은 기계어 변환
