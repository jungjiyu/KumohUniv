: 이진 검색 트리 Binary Search Tree
	: 중복 값은 허용하지 않는다
	: 부모 노드보다 왼쪽 자식 노드는 작고, 오른쪽 자식 노드는 큰 구조
		: 젤 왼쪽 리프 노드는 최솟값, 젤 오른쪽 리프 노드는 최댓값
		: 루트 노드의 왼쪽 영역 노드들은 죄다 루트 노드보다 작고, 오른쪽 영역 노드들은 죄다 루트 노드보다 값이 크다
	: 원소 삽입 방법>> NULL 나올때까지 이동하여 추가
	: 원소 삭제 방법 >>
		case 1. 자식 노드가 없는 노드(== 두 포인터가 모두 NULL)를 삭제 : 
			1. 해당 노드 삭제
			2. 부모노드가 해당 노드를 가리키고 있던 포인터에 NULL 대입

		case 2. 자식 노드가 하나만 있는 노드를 삭제
			1. 임시 노드 temp 로 해당 노드의 자식 노드 가리킴
			2. 해당 노드 삭제
			3. 부모노드가 해당 노드를 가리키고 있던 포인터에 temp 값 대입
		
		case 3. 왼오 자식 노드 모두 있는 노드를 삭제
			1. 삭제 노드의 값을 해당 노드의 왼쪽 후손들 중 최댓값으로 변경
			2. 왼쪽 자식 최댓값을 가지던 노드를 삭제
