명령행 인자 commandline argument

public static void main(String[] args) 에서의 String[ ] args, 즉 String 배열은 콘솔 명령행을 통해 프로그램이 시작될때 추가적으로 넘겨받는 값들을 저장하는 배열.


cmd 상으로 입력하는 법 
	: src 디렉터리 경로에서 해당 패키지 수준까지 들어간 후에 디음의 명령어 실행
		:  java 풀클래스명 인자	
		: ex) java day01/Main "hello"

	: cmd 창 여는 법
		sol1 >> 패키지 우클릭 하고 show in local terminal
			: 바로 해당 경로(ex: C:\FebJava\JSPWorkspace\SchoolJava\src\day01)에서 열려서 편하다
		sol2 >> 걍 컴터의 cmd 창 이용
			: 해당 경로로 직접 이동한 후 명령어 실행

		sol3 >> debug configuration - Argument 
			: 걍 명령어 입력할 필요 없고 바로 인자 입력하면 됨



자바의 경우 일반적인 변수는 초기화 안하면 아예 못쓴다
	: static 필드의 경우 링크의 '준비' 과정에서 디폴트값으로 초기화 되기 때문에 그냥 사용 가능하다
		


부동소수점방식을 사용하기 때문에 소수값의 경우 정확하지 않은 경우가 있다
	: 정확성보단 효율성을 따지기 떄문
		double sum_d = 0.0;
		for(int i = 0; i < 1000000; i++) {
		sum_d += 0.1;
		}
		System.out.println("double:" + sum_d);


정수 리터럴
► 10진수, 8진수, 16진수, 2진수 리터럴
	► 정수 리터럴은 int 형으로 컴파일
15 -> 10진수 15
015 -> 0으로 시작하면 8진수. 십진수로 13
0x15 -> 0x로 시작하면 16진수. 십진수로 21
0b0101 -> 0b로 시작하면 2진수. 십진수로 5
