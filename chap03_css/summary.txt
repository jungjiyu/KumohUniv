선택자 ; 요소"들"을 선택한다
	: 선택자로 선언된 모든 요소들에 적용된다
	: 선택자는 단순 태그만 말하는게 아니라 클래스나 아이디 같은것도 말하는거다
	: 종류
		1. 타입선택자 >> 특정 태그
			: 선택자 그룹 >> , 로 여러개 구분
		2. 전체 선택자(universal selector) >> 
		3. 클래스 선택자(class selector) >>
		4. 속성 선택자(attribute selector) >> 어떤 속성을 가지는 얘들만 찾아낸다
		5. 의사 선택자(pseudo-class) >> 
			: 없는데 있는것처럼 해주는거
