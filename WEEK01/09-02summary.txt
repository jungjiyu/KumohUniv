컴퓨터 구성. 자원 : 시스템버스, RAM , CPU, HDD/SSD(하드디스크), 메모리 >> 
	: 컴퓨터가 돌아가기 위해서는 자원이 필요, 자원은 한정적
	: CPU >> 연산
	: RAM >> 주기억 장치
		: (데이터가 저장되는 공간이 아니고) 처리해야 할 여러 데이터가 RAM에 올라가 CPU에 데이터를 주어 정보 처리를 도와주는 장치
			: 쉽게 말해 SSD에 저장되어있는 각종 운영체제나 프로그램 등이 컴퓨터를 키게 되면 RAM으로 복사가 된다. 그렇게 되면 RAM에서 CPU로 처리할 데이터를 넘겨주어 처리를 하게 된다. 컴퓨터가 꺼지면 RAM에 있는 데이터는 전부 지워지지만 SSD에 있는 각종 자료나 운영체제 프로그램은 지워지지 않고 그대로 있는다
		: 휘발성 o

	: HDD/SSD >> 보조기억장치
		: HDD 하드디스크 >>  문서파일 사진 동영상 등이 저장되는 장치
		: SSD >> HDD보다 빠른 보조기억장치
		: 휘발성 x

	: 버스 >> 데이터를 통신할 수 있게 해주는 통로. 시스템
		: 종류
			System Bus (=Computer Bus) : CPU와 메모리를 연결하는 Subsystem
			I/O Bus : 메모리와 다른 입출력 장치와 통신을 하는 Subsystem
